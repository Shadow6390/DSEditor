Reference

enemiesInRadius

Returns a list containing the instance IDs of all the enemies that are within a certain radius of a position.

Arguments and Return

Argument Name Argument Type Description
x realThe x coordinate of the desired position.
y realThe y coordinate of the desired position.
radius realThe radius to check.

Return

Return Type Description
listA list containing all the enemies within the radius of the position (should call listDestroy when the list is no longer used).
