listCreate

Creates a list. A list has the same behaviour as a dynamic array, which means elements can be added to the list without having control over the maximum size array. When a list is created, memory is allocated for it and its elements. As such, it is important to call listDestroy whenever the list will no longer be used, in order to free the allocated memory and avoid memory leaks.

Arguments

This function does not have any arguments

Return

Return Type Description
RealThe id of the created list.
